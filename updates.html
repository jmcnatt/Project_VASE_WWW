August 14, 2011

Project_VASE Deploy and Connect are both in alpha stage. We are hoping to have both pieces of software available for download on this site and sourceforge. Below are both a screenshot of the software in action.

July 15, 2011

Project_VASE Connect is underway! This will allow students to interface with their virtual machines without remote desktop or VNC. Connect uses the VMRC program from VMware to launch the consoles.

June 28, 2011

The Deployment wizard in VASE Deploy is nearly complete and is now created Object representations of virtual machines from templates. Scripts that clients will use will reside inside the program's root directory, and then copied to the appropriate Virtual Machine when called.

June 23, 2011

Virtual machines will be updated automatically. The user interface is now multi-threaded. Completetion date has been set to July 13, 2011. VASE Deploy will move into pre-alpha stage soon after unit testing is complete.

Brenton is continuing to script Windows Server 2003 installations.

June 21, 2011

Development has begun on the Project_VASE deployment system. The project will use the Java API provided by VMware and available for download on sourceforge.

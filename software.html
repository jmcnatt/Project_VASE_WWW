VASE Deploy

VASE Deploy is a java-based open source application for deploying and managing Virtual Machines loaded in a VMware ESX environment following the Project_VASE design. Features include:

  • Deploy a group of Virtual Machines for a specific team
  • Choose services to install on a Virtual Machine
  • Load pre-configured exploits and user accounts
  • Show the statistics on existing Virtual Machines
System Requirements:
  • Windows XP/Vista/7
  • Java version 5 (version 6 with latest update recommended)
  • Windows Powershell
  • VMware PowerCLI

VASE Connect

VASE Connect is a java-based open source application for connecting to virtual machines in a VMware ESX/ESXi-based environment with a datacenter. Connect utilizes the VMware Java API and VMware VMRC. Features include:

  • Launch one or more virtual machine consoles within a datacenter
  • Full-screen console support
  • Perform power on, power off, suspend, shutdown, restart, and reset operations
  • Observe current power configuration of virtual machines
System Requirements
  • Windows XP/Vista/7
  • Java version 5 (version 6 with latest update recommended)
  • VMware VMRC (included with installer)
